# AQI-India
AQI over the last 10 years

A high-performance, Power BI-style interactive dashboard built to visualize the Air Quality Index (AQI) trends across India over the last decade. This project provides a state-wise analysis of pollution levels using interactive maps, historical trend lines, and regional filters.

🚀 Key Features
:->10-Year Historical Data: Comprehensive tracking of AQI from 2015 to 2025.

:->Interactive India Map: Custom SVG map with dynamic color-coding based on AQI categories.

:->Regional Filtering: Analyze data by North, South, East, West, and Northeast regions.

:->Real-time Rankings: Dynamic leaderboards showing the most and least polluted states.

:->KPI Scorecards: Instant visibility into National Averages, Trends, and Cleanest States.

:->Category Analysis: Distribution charts showing the percentage of "Good" vs "Severe" air quality days.

🛠️ Tech Stack
:->Frontend: HTML5, CSS3 (Modern Glassmorphism UI)

:->Charts: Chart.js (Line, Bar, Donut, and Custom Gauges)

:->Mapping: Custom SVG Path integration for Indian States/UTs.

:->Data Handling: JavaScript-based data processing for instant filtering without page.

📝 How to Use
1.Clone the repository.

2.Open index.html in any modern web browser.

3.Use the Year Slicer at the top to travel through time.

4.Hover over the Map to see specific state growth/decline data.
